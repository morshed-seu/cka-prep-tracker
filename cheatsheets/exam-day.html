Exam-day card — read the night before and the morning of; nothing here is new

Companion to the Week 8 lessons (8.5–8.9 and 8.12 in one page). Print it; it's the only review the final 48 hours need besides your pruned half-page of facts.

The format

  • ~15–20 weighted tasks · 120 min · 66% to pass. Graded on cluster state; partial credit per task.
  • PSI Secure Browser → remote XFCE desktop: terminal and Firefox run remotely. No local bookmarks, aliases, clipboard tools; expect a small latency tax.
  • Use each task's click-to-copy for names/paths — never retype identifiers.
  • Allowed docs, one extra tab: kubernetes.io/docs (+blog), helm.sh/docs, gateway-api.sigs.k8s.io. On-site search fine; leaving those domains is not.
  • Results by email within 24h. A retake is included — that's why you can play it calm.

PSI check-in (30 min early — use all of it)

  • Desk completely clear · one monitor · phone away · alone, door closed.
  • Government ID matching the registration name exactly.
  • Webcam pan: desk, under desk, room — make it boring.
  • Water in a clear container only. No headphones; no talking to yourself.
  • System check re-run the day before, same machine + network. Wired > Wi-Fi.
  • Breaks don't stop the clock and cost re-verification — bathroom before check-in.

First 2 minutes inside

alias k=kubectl
export do="--dry-run=client -o yaml"
export now="--grace-period=0 --force"
# vim: set ts=2 sw=2 et in ~/.vimrc if needed

Open the docs tab. Breathe once. Then task 1.

Per-task ritual — every task

  1. Run the task's use-context line — even if "already there".
  2. Namespace: read it, then -n it (one habit, always).
  3. Node work? ssh in → do it → exit. Glance at the prompt before every block.
  4. Solve → verify → next.

Time strategy (decided; just execute)

  • Pass 1: ≤6 min per task. Stuck or over → flag in the UI, one-word note, move on. Touch every task — unreached = 0, half-done ≠ 0.
  • Pass 2: flags by weight, heaviest first.
  • Last 5 min: sweep flags + verifications. Nothing new; no "improving" working answers.
  • Don't self-grade mid-exam; flags are the system working, not a score.
  • Rattled? Hands off → one breath → re-read the task slowly → context line → continue. Ten seconds.

Final 48 hours

  • Day −2: ≤1h — pruned notes, triage sheet, etcd pair typed once from memory. Logistics sweep (system check, ID, time zone). No new topics, no last mock.
  • Day −1: 20-min flash review before mid-afternoon, then closed. Room prepped to spec today. Ordinary evening, ordinary bedtime.
  • Morning: normal food, optional 20-min glance, fill the (clear) bottle, log in the minute check-in opens.

Facts that must be in your fingers

  • etcd: etcdctl save (online: endpoints + cacert/cert/key) · etcdutl status/restore (offline, fresh dir, repoint hostPath) — card.
  • Upgrade order: kubeadm → plan → apply → per node drain → kubelet → uncordon. CP before workers, one minor at a time.
  • 401 = who are you (credential) · 403 = you may not (RBAC — read the identity it names).
  • Service dead? endpoints → selector/targetPort → triage sheet order.
  • Generators + jsonpath shapes: commands card.
