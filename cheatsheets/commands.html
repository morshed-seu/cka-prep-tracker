kubectl commands — one-page card

Print this (it fits one page) and check your drill runs against it. The lessons behind it are 7.10–7.12 (generators, jsonpath, fast edits) and 7.7–7.8 (RBAC). By exam week you shouldn't need it.

0 · Session setup (first 60 seconds)

alias k=kubectl
source <(kubectl completion bash)
complete -o default -F __start_kubectl k
export do="--dry-run=client -o yaml"
export now="--force --grace-period 0"
export KUBE_EDITOR=vim

Per-task ritual: run the task's use-context line → k config get-contexts (check the NAMESPACE column) → solve → verify → next.

1 · Generators (scaffold, never hand-write)

k run web --image=nginx:1.27 --port=80 --env=A=b $do
k run t --rm -it --image=busybox --restart=Never -- wget -qO- svc
k create deploy web --image=nginx:1.27 --replicas=3 --port=80
k expose deploy web --port=80 --target-port=8080 --type=NodePort
k create service clusterip web --tcp=80:8080   # no workload yet
k create job once --image=busybox -- echo done
k create cronjob tick --schedule="*/5 * * * *" --image=busybox -- date
k create cm cfg --from-literal=K=V --from-file=app.conf
k create secret generic sec --from-literal=PASS=x
k create ns team1 · k create sa app-sa · k create quota q --hard=pods=10

--image before --; everything after -- is the container command. run = Pod, not Deployment.

2 · RBAC one-liners + proof

k create role r --verb=get,list,watch --resource=pods -n ns
k create clusterrole cr --verb=get --resource=nodes,pods/log
k create rolebinding rb --role=r --user=jane -n ns
k create rolebinding rb2 --clusterrole=view \
  --serviceaccount=ns:app-sa -n ns
k create clusterrolebinding crb --clusterrole=cr --group=devs
k auth can-i list pods --as=jane -n ns
k auth can-i --list --as=jane -n ns
k auth can-i get po --as=system:serviceaccount:ns:app-sa -n ns
k auth whoami

--serviceaccount is always ns:name · roleRef immutable → delete + recreate · --as=user drops cert groups: add --as-group=devs.

3 · jsonpath / columns / sort

# one per line → range/end
k get po -o jsonpath='{range .items[*]}{.metadata.name}{"\n"}{end}'
# filter a list
k get no -o jsonpath='{.items[*].status.addresses[?(@.type=="InternalIP")].address}'
# decode a secret key
k get secret s -o jsonpath='{.data.k}' | base64 -d
# current server URL
k config view --minify -o jsonpath='{.clusters[0].cluster.server}'
# several fields → custom-columns
k get po -A -o custom-columns=NAME:.metadata.name,NODE:.spec.nodeName
k get po -A --sort-by=.status.containerStatuses[0].restartCount
k get events --sort-by=.metadata.creationTimestamp

Single-quote the whole path. Fallback that still scores: -o yaml | grep -A5 key.

4 · kubectl explain (offline docs)

k explain pod.spec.containers.livenessProbe
k explain deploy.spec.strategy --recursive | less
k explain networkpolicy.spec.ingress
k explain pv.spec.accessModes
k api-resources | grep -i <name>   # kind, short name, namespaced?

Faster than Firefox for field names/spelling. --recursive shows the whole subtree; grep it for the field you half-remember.

5 · Fast edits & rollouts

k edit deploy web                  # templates: fully mutable
k set image deploy/web nginx=nginx:1.28
k scale deploy web --replicas=5
k rollout status/undo/history deploy/web
# bare pod, immutable field → dump-edit-replace
k get po X -o yaml > f.yaml ; vim f.yaml
k replace --force -f f.yaml

Pod-mutable: image, activeDeadlineSeconds, toleration additions. Everything else → the loop. Controller-owned pod? Edit the controller, not the pod.

⚠ Speed traps

  • Wrong namespace — a context may carry one; check get-contexts before creating anything.
  • cronjob schedule unquoted — the shell eats *; always --schedule="*/5 * * * *".
  • expose without --port fails; create deploy --port sets containerPort only, creates no Service.
  • can-i "no" exits 1 — fine interactively, kills set -e scripts on a correct answer.
  • kubectl edit rejected — your buffer is saved to /tmp/kubectl-edit-*.yaml; don't retype it.
  • Verify everything: k get X after create, rollout status after image change, can-i after RBAC, wget/curl after Service. Partial credit exists; silent typos don't score.
